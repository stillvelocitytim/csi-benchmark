The efficiency frontier

Each bubble is a frontier AI model. The x-axis shows what it costs to run a single benchmark task; the y-axis shows how well the model scores. Bubble size reflects the model’s overall CSI—the bigger the bubble, the more efficiency you get per dollar per second. Models in the top-left corner are the efficient frontier: high capability at low cost.

Where your dollar goes

Each rectangle is sized by its model’s CSI score—the composite measure of capability per second per dollar. Larger rectangles represent models that deliver more useful intelligence for the money. Color indicates the efficiency tier. If you spent $1 on every model, this is roughly how much productive output you’d get back from each.

What does $1 buy?

The same task costs radically different amounts depending on which model you use. We priced two real tasks—summarizing NVIDIA’s 80,000-token 10-K filing, and building a DCF valuation model—across all 16 frontier models using published API pricing. The bars show how many times each model can perform the task for one dollar.
